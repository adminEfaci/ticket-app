Testing Workflow

  1. Click on Login Page above
  2. Sign in with the admin credentials
  3. Navigate to Upload page
  4. Upload an XLS file from the samples folder
  5. System will parse tickets and match to clients
  6. Optionally upload the corresponding PDF for image extraction
  7. View results in the dashboard
Note: The system uses fuzzy matching to automatically assign tickets to clients based on reference patterns. All 276 clients from your CSV have been imported with their respective rates.
